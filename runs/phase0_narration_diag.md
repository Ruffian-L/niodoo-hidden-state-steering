## ROOT CAUSE — CONFIRMED 2026-07-01 evening

**The public release rewrote the runtime system prompt, and the rewrite un-taught the model the control language.**

Chain of evidence:
1. Bisect: e9394e8 narrates identically to HEAD (0 markers, 14MB) → not the RC/BC commits.
2. The known-good evidence (`evidence/correct/…`) landed fully-formed in the FIRST public commit c3af082 — it was generated by the **team_build-era binary**, before the public tree existed.
3. Diff of `default_runtime_system_prompt()` (team_build Jun-8 snapshot vs public): team_build's "Cognitive Mirror" prompt commands *"Emit the exact tag line when needed. Do not explain why. [REQUEST: LOCK] … stop cleanly."* The public "INTERNAL MONITOR" prompt says *"Visible emission is optional — the runtime detects them whether or not they appear."* At temp 0, Llama-8B takes the offer: no tags emitted, ever.
4. **Falsification run (decisive):** HEAD binary, seed 42, temp 0, identical flags, `--system-prompt-file` = team_build Cognitive Mirror text → `VISIBLE ANSWER: 3` (correct) + `[REQUEST: LOCK]` emitted. Same binary with the public prompt: 0 markers, 0 tags. **Only the prompt changed. The binary was never broken.**

No tags → nothing for the parser → no LOCK → finalization taper never fires → model narrates to the 256-step cap. "Niodoo narrating himself" = the control language went silent.
